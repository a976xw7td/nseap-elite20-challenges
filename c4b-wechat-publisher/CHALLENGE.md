# Challenge C4B：公众号文章生成技能（WeChat Article Pipeline Skill）

> 🔥 C4 的另一条进阶路线——不是评审技能，而是**创造内容发布流水线**。
> 本质：把 Doc-Pipeline 的"格式转换"能力提取出来，变成你自己的公众号文章生成技能。

---

## 背景

微信公众号是中国最重要的内容发布平台之一。但从一篇 Markdown 或 Word 文档到一篇排版精美的公众号文章，中间有一系列技术坑：

- 公众号编辑器**只接受 inline CSS**——不能用 `<style>` 标签、不能用 class
- `<h1>` 被禁止——要降级为 `<h2>`
- `<div>` 被禁止——要用 `<p>` 代替
- `<script>` 和 `<iframe>` 被禁止——所有交互性内容失效
- 图片必须走公众号自己的 CDN——外部链接图片会丢失
- SVG 在保存时会消失——只有 PNG 能存活

现有的 `wechat-html-converter` 技能已经能做基础转换，`wechat-math-html` 能处理数学公式。但它们是**通用工具**——没有针对"你的内容"做个性化定制。

**C4B 的任务：拿到这个 starter kit，用 skill-creator 把它改造成你自己的公众号文章生成技能，并用它发布一篇真实的公众号文章。**

## 挑战目标

```
Markdown / Word 文档
       ↓
  ① 读取源文件内容
       ↓
  ② 转换为 HTML
       ↓
  ③ 应用公众号排版规则（inline CSS、标签过滤、样式美化）
       ↓
  ④ 输出 copy-paste ready 的 HTML
       ↓
  ⑤ 发布到微信公众号（手动粘贴即可）
```

你需要做的不只是"让转换跑通"——而是让输出的文章**好看、有个性、可定制**。

## 四级任务（由浅入深）

### Level 1：跑通 + 微调（Bronze 🥉）

> **安装 starter kit，成功转换一篇文章，微调样式参数。**

- 安装 starter kit，跑通 `convert_to_wechat.py`
- 用你自己的一篇 markdown 文章测试
- 修改 `wechat_styles.md` 中的样式参数（颜色、字号、间距）
- 在浏览器中预览输出效果

**交付：** 能跑通的 starter kit + 你调整过的样式

---

### Level 2：用 skill-creator 改造（Silver 🥈）

> **使用 skill-creator 技能，把 starter kit 改造成你的个性化版本。**

核心步骤：

1. 对 Claude 说：`"帮我用 skill-creator 改造这个 WeChat 公众号技能"`
2. skill-creator 会引导你定义：技能做什么、触发条件、输入输出、测试用例
3. 至少新增**两项能力**（从上面"缺少的能力"中选，或自己想新的）
4. 用 skill-creator 的 eval 流程测试你的技能

**推荐新增能力（任选 2+）：**

| 能力 | 难度 | 价值 |
|------|------|------|
| 多主题支持（default/accent/minimal） | ★★ | 让文章有不同风格 |
| callout 框（📘知识/🔧实践/⚠️注意） | ★★ | 公众号文章的核心排版元素 |
| 自动目录 | ★★ | 长文必备 |
| 文章元数据（标题/作者/日期/摘要） | ★ | 完善文章结构 |
| 页脚版权声明 | ★ | 实际发布必需 |
| 中文排版优化（间距/标点） | ★★★ | 专业感提升 |
| 数学公式支持 | ★★★★ | 技术含量高，参考 wechat-math-html |

---

### Level 3：发布真实文章（Gold 🥇）

> **用你的技能生成一篇真实的公众号文章，发布到微信公众号。**

这是最关键的一步——技能好不好，用了才知道。

1. 选择一个你想写的主题（可以是你的 C4 技能说明、学习心得、技术教程……）
2. 用 Markdown 或 Word 写好内容（AI 辅助）
3. 用你的技能转换为公众号 HTML
4. 在浏览器中预览，调整到满意
5. 粘贴到公众号编辑器 → 预览 → 发布
6. 把文章链接分享到群里

**没有公众号怎么办？**
- 注册一个个人公众号（免费，10 分钟搞定）
- 或者用同学的公众号联合发布
- 或者用公众号草稿箱截图证明（不发布也行，但分数打折）

---

### Level 4：完整发布流水线（Platinum 💎）

> **实现从"一个想法"到"一篇发布的文章"的全自动流水线。**

在 Level 3 基础上，增加：

- **写作辅助集成**：技能不仅转换格式，还能帮你从大纲生成初稿
- **多格式输出**：同一内容 → 公众号 HTML + 知乎 Markdown + 纯文本版本
- **模板系统**：预设多种文章类型模板（技术教程、学习心得、产品介绍……）
- **SEO/阅读体验优化**：自动加标题摘要、段落分隔、重点高亮
- **一键发布工作流文档**：记录从写作到发布的完整 SOP

## 起步指南

### Step 1：安装并跑通 starter kit（Day 1）

```bash
# 解压 starter kit
unzip c4b-wechat-publisher-starter.zip

# 安装依赖
pip install markdown beautifulsoup4 python-docx lxml --break-system-packages

# 跑一遍示例
python scripts/convert_to_wechat.py examples/sample_article.md output.html

# 在浏览器中打开 output.html 查看效果
```

### Step 2：理解代码结构（Day 1–2）

读 `convert_to_wechat.py`，理解：
- `read_markdown()` — 怎么把 Markdown 变成 HTML？
- `read_docx()` — 怎么从 Word 提取内容？
- `sanitize_html()` — 怎么过滤违规标签？
- `apply_inline_styles()` — 怎么加 inline CSS？
- `WECHAT_STYLES` — 默认样式在哪定义？

### Step 3：用 skill-creator 改造（Day 2–5）

对 Claude 说：

> "我有一个 WeChat 公众号文章生成的 starter kit，帮我用 skill-creator 把它改造成我自己的版本。我想加入 [你想要的能力]。"

skill-creator 会引导你：
1. 定义技能目标 → 2. 讨论边界情况 → 3. 写 SKILL.md → 4. 创建测试用例 → 5. 运行 eval → 6. 迭代改进

### Step 4：写一篇文章并发布（Day 5–7）

1. 选题（建议写你的 Elite20 学习经验或 C4 技能介绍）
2. 用 AI 写初稿（Markdown 格式）
3. 用你的技能转换
4. 预览 → 调整 → 满意
5. 粘贴到公众号编辑器 → 发布
6. 截图 + 链接发群里

### Step 5：提交

按命名规范打包提交。附上 skill-creator 的完整使用日志（这是评审重点之一）。

## 为什么 C4B 值得做

```
C4  = 我会做一个技能
C4B = 我会做一个内容发布流水线
```

这是从"工具制造者"到"内容生产力系统构建者"的跃迁。

在真实世界里，**内容即产品**。一个能快速把想法变成精美发布内容的人，在任何组织里都是稀缺资源。C4B 训练的是：

1. **拿来主义实战**——不是从零写代码，而是理解→改造→超越已有工具
2. **元技能应用**——用 skill-creator（做技能的技能）做技能（内容发布技能）——这是 Agent 系统的递归本质
3. **端到端验证**——不是"能跑"就行，而是"发布了真实文章"才算数

完成 C4B 后，你拥有的不只是一个技能——而是一个**个人内容发布引擎**。以后写任何文章，一条命令就能变成排版精美的公众号。
